### 🌍 Overview
**EcoCircuit** is a specialized platform designed to tackle the growing global issue of electronic waste. It bridges the gap between individual users and certified recycling centers, providing a structured workflow for discarding old gadgets while ensuring environmental compliance and data security.

### 🛠️ Tech Stack
| Component | Technology |
| :--- | :--- |
| **Frontend** | React.js / Vite |
| **Backend** | Spring Boot / Node.js |
| **Database** | MySQL / PostgreSQL |
| **Authentication** | JWT (JSON Web Tokens) |
| **Styling** | Tailwind CSS / CSS3 |

### ✨ Key Features
*   **Device Categorization:** Intelligent sorting of waste into categories (Large Appliances, IT/Telecom, Consumer Electronics).
*   **Pickup Scheduling:** Integrated booking system for users to arrange e-waste collection.
*   **Recycling Center Locator:** Map-based search to find the nearest authorized e-waste drop-off points.
*   **Impact Dashboard:** Visual metrics showing the amount of heavy metals (Lead, Mercury, etc.) diverted from landfills.
*   **Admin Panel:** Full control over logistics, verification of recycling certificates, and user management.
